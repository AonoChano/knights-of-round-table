# Stage Summary Projection

## 适用场景

当系统需要把内部讨论转换成可见思考摘要时使用。

## 执行步骤

1. 读取当前阶段的内部结果
2. 提炼本阶段真正推进了什么
3. 用第一人称表述思考进展
4. 隐藏参与者身份、原始措辞和多模型事实
5. 生成标题、短摘要、详细说明和树节点

## 输出要求

- 不复制原始内部文本
- 不暴露“多个模型在讨论”
- 只输出用户界面需要的安全字段
